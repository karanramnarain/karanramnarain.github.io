Work

Een overzicht van projecten waar ik als onderdeel van mijn studie aan Software Engineering aan heb meegewerkt.

Fibonacci - Slimme gevels voor duurzame gebouwen

Een studieproject aan UNASAT gericht op een innovatieve, duurzame oplossing voor het groeiende probleem van warmte- en energieverbruik in gebouwen. Het project draait om een slimme gevel die automatisch reageert op omgevingsfactoren zoals zonlicht, temperatuur en luchtkwaliteit.

Mijn bijdrage lag vooral bij de technische kant: ik heb gewerkt aan het aansturen van de machine met de Arduino, zodat deze beweegt en draait en heb de C++ code hiervoor aangepast. Daarnaast heb ik meegeholpen om de constructie in elkaar te zetten.

  • Arduino
  • C++
  • Constructie

MedixFlow

Een webapplicatie waarmee afspraken binnen de afdeling kaakchirurgie kunnen worden geregistreerd en beheerd. De applicatie ondersteunt twee rollen: patiënt en administrator.

Ik heb meegewerkt aan het gedeelte waar de client (patiënt) afspraken kan maken via de website, inclusief het inlogsysteem.

  • Node.js
  • Login systeem
  • Afspraken maken
  • Webapplicatie
